## ![Installation](https://img.shields.io/badge/-Installation-94e2d5?style=for-the-badge)

Quick setup to get PurrPrompt running in your terminal:

```bash
# 1. Clone the repository
git clone https://github.com/AKRiLLiCK/purrprompt.git && cd purrprompt

# 2. Source in your .bashrc
echo "source $(pwd)/.bashrc" >> ~/.bashrc

# 3. Reload your shell
source ~/.bashrc
```

> [!NOTE]
> Requires a [Nerd Font](https://www.nerdfonts.com/) for icons and glyphs to render correctly
